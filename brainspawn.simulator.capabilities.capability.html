brainspawn.simulator.capabilities.capability module¶

Abstract base class for Capabilites Essentially a ‘data type’ Capability represents observable data that a given object in a nengo model has to offer. The capablity defines how to connect an observer node to the object to be observed in order to collect the data. It also maintains some metadata regarding the type of data offered, including determining the dimensions of the given data from the given object.
