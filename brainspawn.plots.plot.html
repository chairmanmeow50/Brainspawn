brainspawn.plots.plot module¶

Abstract base class for plots.

class brainspawn.plots.plot.Plot(main_controller, nengo_obj, capability)[source]¶

Bases: plots.base_plot.BasePlot

Plot class. In order to add plots to Brainspawn, you will want to inherit from this class.

Note that subclasses must call the base class constructor.

Sets default config values for all plots The values contained in this dictionary are used to configure the plot.

For convenience in title string formatting, we set ‘TARGET’ and ‘DATA’ to default values of the target object, and represented data, respectively.
